========================================
  ゲームBGM配置フォルダ
========================================

このフォルダに .ogg ファイルを配置してください。

ファイル名の例:
- battle1.ogg
- battle2.ogg
- battle3.ogg

注意事項:
1. ファイル名は sounds.json に登録されている必要があります
2. ファイル名にスペースや特殊文字を使用しないでください
3. Ogg Vorbis 形式のみサポートされています

新しい曲を追加する場合:
1. このフォルダにOGGファイルを配置
2. sounds.json に以下のようなエントリを追加:

"gamebgm.曲名": {
  "sounds": [
    {
      "name": "gamebgm/曲名",
      "stream": true
    }
  ]
}
